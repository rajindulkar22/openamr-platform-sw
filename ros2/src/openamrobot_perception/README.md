# openamrobot_perception

Planned ROS 2 package for perception modules and perception pipelines.

Expected future responsibilities:

- camera processing
- object detection
- fiducial marker detection if used globally
- obstacle perception
- point cloud processing
- perception-related launch and configuration files

Docking-specific perception logic may remain in `openamrobot_docking` if it is used only for docking.

This package should not contain Nav2 configuration, robot description files, Gazebo worlds, or hardware drivers.

Status: planned placeholder.
