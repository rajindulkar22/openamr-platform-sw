# openamrobot_drivers

Planned ROS 2 package for hardware driver integration.

Expected future responsibilities:

- sensor driver integration
- motor controller communication bridges
- battery monitor interfaces
- embedded controller communication
- serial, CAN, RS485, or other hardware communication nodes
- driver launch and configuration files

This package should not contain embedded firmware source code. Firmware belongs in the separate `openamr-platform-fw` repository.

Status: planned placeholder.
