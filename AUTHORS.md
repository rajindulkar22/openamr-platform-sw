# Contributors

Contributors are recognized through:

- Git history
- Pull requests
- Issues
- Discussions
- Documentation
- Reviews
- Testing
- Community support

Contributors may add themselves to this file through a pull request.

Suggested format:

```text
- Name or GitHub handle — contribution area
```

Examples:

```text
- username — ROS 2 simulation
- username — Nav2 integration
- username — autodocking
- username — documentation
- username — embedded firmware integration
```
